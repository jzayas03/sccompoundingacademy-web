# SCCA Landing Page Components

This package contains separate plug-and-play HTML/CSS components for the Santa Cruz Compounding Academy landing page.

## Files
- `index.html` — full demo page using all components
- `styles.css` — global CSS design system
- `components/` — individual HTML sections
- `assets/` — SVG logo and icons

## How to use
1. Copy `styles.css` into your project.
2. Copy the HTML section you need from `/components`.
3. Keep the `/assets` folder path or update image/logo references.
4. Replace placeholder copy, images, phone number, email, and program details.
